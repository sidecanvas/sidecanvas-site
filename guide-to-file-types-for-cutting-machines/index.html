Guide to File Types for Cutting Machines: PNG, SVG, EPS, DXF

Not sure which cut file type you need to download for your next craft project? This guide will give you a quick rundown!

File types used for cutting plotters

PNG File Type

  • Stands for Portable Network Graphics
  • Can be used to cut with Cricut and Silhouette machines

SVG File Type

  • Stands for Scalable Vector Graphics
  • Best for most Cricut machines

DXF File Type

  • Stands for Drawing Interchange Format
  • Best for most Silhouette machines

EPS File Type

  • Stands for Encapsulated Postscript
  • This is used in older CNC cutting machines
